## 2026-07-01 Update: Stage Clear Reward Selection Timing

- Build-changing attack and skill rewards should be offered at Stage Clear, not after every Room Clear in the long-term structure.
- Stage Clear reward cards appear before returning to the Stage Map.
- The selected reward is applied to the current Run before connected next Stage candidates are shown.
- This means the player chooses the next Stage with the updated build already active.
- Current Room Clear rewards remain an MVP reward UI/effect test pipeline until Stage Clear rewards are implemented.

## 2026-07-02 Update: First Elemental Build Axis - Katana + Lightning

The first elemental build axis is planned as Katana + Lightning.

This is a planning direction, not an implemented system.

Combat rule direction:

- The current left-click 1/2/3 combo remains the base combat loop.
- Lightning should first attach to the 3rd combo hit because the 3rd hit already feels like the combo payoff point.
- `체인 라이트닝` is the first identity test for Lightning, even though it is a Gold-grade design candidate.
- `전기 스파크` should act as an early/Bronze entry card, not a Silver-only reward.
- `감전 칼날` should start as a minimal short Shock mark, not a full status-effect framework.
- `과충전` should connect dodge and the next attack.
- `폭풍검` is the first representative Prism payoff candidate after Chain Lightning feels good.

Recommended implementation order when this moves to code:

1. Chain Lightning MVP from confirmed 3rd combo hit.
2. Simple Electric Spark entry reward.
3. Dodge overcharge reward.
4. Minimal Shock mark.
5. Storm Sword Prism payoff.

Do not build every element, every Lightning card, or the full Prism system at once.

Detailed card lists and Prism unlock rules live in `Docs/ElementalRewardSystem.md`.
